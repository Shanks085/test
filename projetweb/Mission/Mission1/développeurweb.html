Un développeur Web est un programmeur qui développe des applications Web en utilisant un modèle client-serveur. Les applications utilisent généralement HTML, CSS et JavaScript dans le client, et n'importe quel langage de programmation à usage général dans le serveur. HTTP est utilisé pour les communications entre le client et le serveur.
